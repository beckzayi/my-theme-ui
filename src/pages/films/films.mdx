# Read Films

This method will return data of all films from a Site. Pay close attention to query string filters and pagination in order to control responses from the server.

Check [Read a Film](/films/get-film) if you need data of a single film.

## Try it out

Click the button below to open a new browser to play with GraphiQL, which prefills query string. Ask for what you want, and get predictable results.

## Code Example

```graphql
query {
  allFilms {
    id
    title
    createdAt
    director
  }
}
```
