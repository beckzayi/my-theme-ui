# Read a Film

This method will return data of one single from a Site. Pay close attention to query variables in order to control responses from the server.

Check [All Film](/films/films) if you need data of all films.

## Try it out

Click the button below to open a new browser to play with GraphiQL, which prefills query string and variables. Ask for what you want, and get predictable results.

<Link query={queryString} />

Remember to fill the query string below in the `QUERY VARIABLES` pane.

```json
{
  "id": "cj0nxmy3fga5s01148gf8iy3c"
}
```
